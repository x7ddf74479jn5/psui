# Installation

## General Installation

このガイドでは、Tailwind が設定された React プロジェクトがすでにあることを前提としています。

1. `pnpm add @x7ddf74479jn5/psui`と`pnpm add -D @x7ddf74479jn5/tw-plugin-psui`を実行します。
2. tailwind.config.js`を修正し、インクルードします：

- `"./node_modules/@x7ddf74479jn5/psui/dist/**/*.{js,mjs}",`を`content`配列の中へ
- `require("@x7ddf74479jn5/tw-plugin-psui")`を`plugins`配列の中へ

```js
module.exports = {
  content: [
    "./pages/**/*.{js,ts,jsx,tsx,md,mdx}",
    "./components/**/*.{js,ts,jsx,tsx}",
    "./node_modules/@x7ddf74479jn5/psui/dist/**/*.{js,mjs}", // この行は重要です！これを入れないと、PSUIのスタイルがパージされます
    ,
  ],
  plugins: [
    require("@x7ddf74479jn5/tw-plugin-psui"), // 重要！これにより、CSS変数を消費するようにtailwindテーマを拡張します。
  ],
};
```

CSS 変数をグローバル css ファイルに追加します。[Concepts/Colors](?path=/docs/concepts-colors--docs)を参考にしてください。

`root`の後に置いた最初のテーマがデフォルトで使用されます。もしあなたのサイトが複数のテーマを使用しないのであれば、これだけで十分です。複数のテーマを切り替えるには、javascript を使って `<html>` の `data-theme` 属性を更新すればよい。

## Remix

Remix でインストールする場合、1 つのステップが追加されます。

`remix.config.js` で `serverDependenciesToBundle` オプションを追加する必要があります。詳しくは[こちら](https://remix.run/docs/en/v1/pages/gotchas#importing-esm-packages)をご覧ください。

```js
/** @type {import('@remix-run/dev').AppConfig} */
module.exports = {
  future: {
    unstable_tailwind: true,
  },
  ignoredRouteFiles: ["**/.*"],
  serverDependenciesToBundle: ["@x7ddf74479jn5/psui"],
};
```
